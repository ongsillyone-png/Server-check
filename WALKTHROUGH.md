# WALKTHROUGH.md — คู่มือการใช้งานระบบ Server Check

---

## 1. เข้าสู่ระบบ (Login)

1. เปิด Browser ไปที่ `http://yourdomain.com`
2. กรอก Username และ Password
3. กด **Sign In**

---

## 2. หน้า Dashboard

แสดงสรุปภาพรวมของระบบ:
- **Cards**: จำนวน Room, Rack, Server, VM และสรุป Inspection วันนี้/เดือนนี้/ปีนี้
- **กราฟ**: แนวโน้ม 30 วัน และ 12 เดือน (Apache ECharts)
- **Pie Chart**: สัดส่วน Pass / Fail / N/A

---

## 3. เดินตรวจตามรอบ (Walkthrough Inspection)

1. คลิก **"เริ่มเดินตรวจตามรอบ"** ใน Sidebar
2. กด **"เริ่มรอบการตรวจใหม่"**
3. เลือก **ห้อง (Room)** ที่ต้องการตรวจ
4. เลือก **ตู้แร็ค (Rack)** ในห้องนั้น
5. เลือก **เครื่อง Server** ที่ต้องการตรวจ
6. กรอก **Checklist**: ตอบรายการตรวจแต่ละข้อ (Pass / Fail / N/A)
7. แนบ **รูปถ่าย** (ถ้ามี)
8. กรอก **หมายเหตุ** (ถ้ามี)
9. กด **"บันทึกผล"** → ไปอุปกรณ์ถัดไปโดยอัตโนมัติ
10. ทำซ้ำจนครบทุก Server ในแต่ละ Rack
11. เมื่อตรวจครบทุก Room กด **"ปิดรอบการตรวจ"**

> 💡 สามารถเดินตรวจจาก **มือถือ** ได้โดยเปิดเบราว์เซอร์ Sidebar จะปิดเป็น Panel slide-in อัตโนมัติ

---

## 4. ดูประวัติการตรวจย้อนหลัง

1. คลิก **"รายงานและประวัติย้อนหลัง"** ใน Sidebar
2. ใช้ Filter ค้นหาด้วย: วันที่, ห้อง, Rack, Server, ผู้ตรวจ
3. คลิกรายการเพื่อดูรายละเอียด Checklist และรูปถ่าย
4. กด **"Export CSV"** เพื่อดาวน์โหลดข้อมูล

---

## 5. ระบบออกรายงาน

คลิก **"ระบบออกรายงาน"** ใน Sidebar:

| รายงาน | คำอธิบาย |
|---|---|
| รายวัน | สรุปผลตรวจประจำวัน |
| รายเดือน | สรุปผลตรวจรายเดือน |
| รายปี | สรุปผลตรวจรายปี |
| Server Summary | ผลตรวจแยกตาม Server |
| Rack Summary | ผลตรวจแยกตาม Rack |
| Room Summary | ผลตรวจแยกตาม Room |
| User Summary | ผลตรวจแยกตามผู้ตรวจ |

---

## 6. แดชบอร์ดผู้บริหาร (Analytics & KPI)

คลิก **"แดชบอร์ดผู้บริหาร (KPI)"** ใน Sidebar:
- Top Server/Rack/Room ที่มี Fail สูงสุด
- Inspection Rate และ Completion Rate
- KPI รายเดือน รายไตรมาส รายปี
- กราฟแนวโน้มการตรวจ

---

## 7. ตั้งค่าการแจ้งเตือน LINE

1. คลิก **"ตั้งค่าระบบ (Settings)"** ใน Sidebar
2. กรอก **LINE Client Key** และ **LINE Secret Key** จาก MOPH Notify
3. ตั้งค่า **เวลาที่ต้องการให้แจ้งเตือนสรุปประจำวัน** (เช่น 08:00)
4. กด **"บันทึก"**

> ระบบจะส่งสรุปผลการตรวจ (ตรวจแล้ว/ยังไม่ตรวจ) ทาง LINE ตามเวลาที่กำหนดทุกวันอัตโนมัติ

---

## 8. การจัดการสินทรัพย์

| เมนู | คำอธิบาย |
|---|---|
| Rooms | เพิ่ม/แก้ไข/ลบ ห้องเซิร์ฟเวอร์ |
| Racks | เพิ่ม/แก้ไข/ลบ ตู้แร็ค (ระบุห้องที่ตั้ง) |
| Servers | เพิ่ม/แก้ไข/ลบ เครื่อง Server จริง (ระบุตำแหน่ง U) |
| VMs | เพิ่ม/แก้ไข/ลบ เครื่อง Virtual Machine |
| Templates | สร้าง Checklist Template |
| Items | เพิ่มหัวข้อใน Checklist |

---

## 9. การจัดการผู้ใช้งาน

| เมนู | คำอธิบาย |
|---|---|
| Users | เพิ่ม/แก้ไข/ลบ ผู้ใช้งาน กำหนด Role |
| Roles | จัดการบทบาท (admin / inspector / viewer) |
